# talent_olympics

## Technologies Used
- Node
- React
- Solana
- Pyth
- Jupiter Exachange API
- Express (Backend)
- Socket IO (For real time changes in SOL/USD prices)
- Shadcn (for UI)
- Zustand

## Setup
- run "yarn" in /backend
- Update env (check sample env)
- run "yarn dev" in /backend

- run "yarn" in /wallet
- Update env (check sample env)
- run "yarn dev" in /wallet
